# 🛠️ Meus Projetos de Arduino

Repositório pessoal de estudos e experimentos com **ARDUINO** ou **ESP32**

Este espaço foi criado para organizar todos os circuitos e códigos que desenvolvo apenas para encher meu GitHub.

## 📂 Estrutura do Repositório

O repositório está organizado de forma que cada pasta representa um projeto único:

*   **Nome da Pasta**: Dentro de cada pasta, você encontrará o código-fonte e os esquemas necessários.
*   **Arquivo `explica.txt`**: Em cada pasta, incluí um arquivo de texto detalhando oque o projeto faz.

## Stack's

*   C++ (Linguagem base do Arduino)
*   Arduino IDE
*   Componentes eletrônicos diversos (Sensores, Atuadores, Displays, etc.)
